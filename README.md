# SudanMed — منصة المستلزمات الطبية 🏥

منصة التجارة الإلكترونية الطبية الأولى في السودان

## الصفحات المتاحة

| الصفحة | الرابط |
|--------|--------|
| الرئيسية | `/` |
| المنتجات | `/products` |
| السلة | `/cart` |
| الدفع | `/checkout` |
| تتبع الطلب | `/track` |
| تسجيل الدخول | `/auth/login` |
| إنشاء حساب | `/auth/register` |

## التقنيات
- **Frontend:** Next.js 14 + TypeScript + Tailwind CSS
- **RTL:** دعم كامل للعربية
- **Font:** Cairo (Google Fonts)

## التشغيل المحلي
```bash
cd apps/web
npm install
npm run dev
```

## طرق الدفع المدعومة
- بنكك (يدوي + رفع إيصال)
- كاش عند التسليم
- تحويل بنكي
- Stripe (بطاقات دولية)

## الترخيص
